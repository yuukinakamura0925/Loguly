## 1. エグゼクティブサマリー

### 本提案の概要

**Loguly（ログリー）** は、研修動画やeラーニングコンテンツの視聴を**確実に完了させる**ための動画視聴ログ管理システムです。

| 項目 | 内容 |
|------|------|
| **解決する課題** | 動画の「見たふり」「早送り視聴」の防止 |
| **主要機能** | スキップ制限付きプレイヤー、秒単位の視聴ログ、進捗管理 |
| **初期費用** | 30万円〜40万円（税別） |
| **月額費用** | 約2,000円〜7,000円 |
| **開発期間** | 約1.5〜2ヶ月 |

**従来のLMS（学習管理システム）と比較して、低コストで導入可能です。**

## 3. 解決策：Logulyのご提案

### 3.1 Logulyとは

**Loguly**は、「**確実に視聴させる**」ことに特化した、シンプルかつ強力な動画視聴ログ管理システムです。

```
┌─────────────────────────────────────────────────────────┐
│                    Logulyの特長                          │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  ✅ スキップ・早送り禁止で「見たふり」を防止              │
│                                                         │
│  ✅ 1秒単位で視聴ログを記録                             │
│                                                         │
│  ✅ 誰がどの動画を何％見たかリアルタイムで把握           │
│                                                         │
│  ✅ 低コスト（使った分だけの従量課金）                   │
│                                                         │
│  ✅ シンプルな操作性（ITに詳しくなくても使える）         │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### 3.2 スキップ制限の仕組み

```
【従来の動画プレイヤー】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
▶ ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
                     ↑ どこでも自由にシーク可能

【Logulyのプレイヤー（初回視聴時）】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
▶ ━━━━━━━━━●━━━━━━━━━━━━━━━━━━━━━━━━━━━
   視聴済み区間  │  未視聴区間（シーク不可）
   （巻き戻し可）│
                │
         現在位置より先には進めない
```

- **初回視聴時:** 視聴した位置より先へのスキップを禁止
- **視聴完了後:** 自由にシーク可能（復習用途）
- **途中離脱しても:** 次回アクセス時に続きから再開

## 4. システム機能一覧

### 4.1 ユーザー向け機能

| 機能 | 説明 | 特長 |
|------|------|------|
| **スキップ制限プレイヤー** | 未視聴区間への早送りを禁止 | 確実な視聴を保証 |

| **カテゴリ別一覧** | 動画をカテゴリごとに整理表示 | 目的の動画を見つけやすい |

### 6.2 採用技術

| 項目 | 技術 | 選定理由 |
|------|------|---------|
| フロントエンド | Next.js | 高速表示、SEO対応 |
| バックエンド | Supabase | 認証・DB・APIを一括提供 |
| 動画配信 | Cloudflare Stream | 高品質配信、セキュリティ |
| ホスティング | Vercel | 高い安定性、自動スケーリング |

### 6.3 セキュリティ対策

| 対策 | 内容 |
|------|------|
| **通信暗号化** | 全通信をHTTPS（TLS 1.2以上）で暗号化 |
| **認証** | メール/パスワード認証、セッション管理 |
| **動画保護** | 署名付きURL（有効期限付き）で不正アクセス防止 |
| **権限管理** | ユーザー/管理者の明確な権限分離 |

## 11. よくあるご質問（FAQ）

### Q1. 動画をダウンロードされて悪用されませんか？

**A.** Cloudflare Streamの「署名付きURL」を使用するため、動画の直接ダウンロードは困難です。URLには有効期限があり、期限切れ後はアクセスできなくなります。

### Q2. スマートフォンでも視聴できますか？

**A.** はい。PC、タブレット、スマートフォンすべてに対応したレスポンシブデザインを採用しています。

### Q3. 既存の動画ファイルをそのまま使えますか？

**A.** はい。MP4、MOV、MKV、AVIなど主要な動画形式に対応しています。アップロード後、自動的に配信に適した形式に変換されます。

### Q4. ユーザー数に上限はありますか？

**A.** システム上の上限はありません。ユーザー数に応じてクラウドサービスの利用料が変動する従量課金制です。

### Q5. 途中で動画を追加・削除できますか？

**A.** はい。管理画面から自由に動画の追加・削除・カテゴリ変更が可能です。

### Q6. 視聴データはどのくらいの期間保存されますか？

**A.** 標準では無期限で保存されます。データ削除のタイミングは運用ポリシーに応じて設定可能です。

### Q7. 社内ネットワーク限定で使用できますか？

**A.** 基本構成はインターネット経由ですが、IPアドレス制限やVPN経由のみでのアクセス制限も追加対応可能です（要相談）。

### Q8. 納品後のサポートはありますか？

**A.** 納品後1ヶ月間は無償でバグ修正対応いたします。継続的な保守契約についてはご相談ください。
